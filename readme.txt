== Description ==
This plugin will allow administrators to chek posts against copyscape via the copyscape API. The plugin allows for the selection of several different plagiarism searches. Copyscape currently supports two kinds of searches. A URL search and a Text search. Both options are implemented in this plugin.

== Installation ==
This section describes how to install the plugin and get it working.

1. Upload the "copyscape" folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
